* 💻 Featured projects

## 🛠️ Built With

* React.js
* Vite
* Tailwind CSS
* Framer Motion
* Lucide React

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/<your-username>/shriyut-portfolio.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
